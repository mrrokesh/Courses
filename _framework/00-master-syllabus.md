**Philosophy:** Learn → Build → Deploy → Get Hired  
**Methodology:** 70% hands-on labs + 30% theory | Weekly mini-projects | Monthly capstones | GitHub portfolio mandatory

> For duration tiers, assessment weights, and weekly rhythm see [Duration Framework](00-duration-framework.md). For package bundles see [Combo Packages](00-combo-packages.md). For project levels L1–L5 see [Project Progression](00-project-progression.md).

## Duration Framework (Applies to All Technical Tracks)

| Duration | Hours | Depth | Projects | Outcome Level |
|----------|-------|-------|----------|---------------|
| **2 Weeks** (Crash) | 40–50 hrs | Awareness + core syntax | 1 mini project | Can build basic apps; needs mentorship |
| **1 Month** | 80–100 hrs | Foundation + workflow | 2 guided projects | Junior-ready basics |
| **3 Months** | 250–300 hrs | Intermediate + integration | 3–4 projects | Internship-ready |
| **6 Months** | 500–600 hrs | Advanced + deployment | 5–6 + 1 capstone | Job-ready (entry–mid) |
| **9 Months** (Advanced) | 750–900 hrs | Expert + specialization | 8–10 + 2 capstones | Industry-ready + certification prep |

**Weekly structure (all durations):**
- Mon–Thu: Concept + live coding lab
- Fri: Mini-project / assignment submission
- Sat: Code review + doubt clearing
- Sun: Revision + documentation (GitHub README)

**Assessment:** Weekly quiz (20%) | Lab completion (40%) | Project (30%) | Portfolio (10%)

# TRACK 1: FULL STACK DEVELOPMENT

*HTML/CSS → JS/TS → React → Node/Express → Spring Boot → REST APIs → Auth → Payments → Deploy*

**Detailed outline:** [tracks/01-full-stack-development.md](tracks/01-full-stack-development.md)

### 2 Weeks — Crash Course

| Week | Topics | Hands-On Project |
|------|--------|------------------|
| W1 D1–2 | HTML5 semantic tags, CSS Flexbox/Grid, responsive design | Personal landing page |
| W1 D3–4 | JavaScript basics, DOM, events, fetch API | Interactive To-Do App |
| W1 D5–7 | React basics: JSX, props, state, hooks | React portfolio site |
| W2 D1–3 | Node.js + Express, REST API, CRUD | REST API for notes app |
| W2 D4–5 | MongoDB/MySQL basics, API + DB integration | Full-stack notes app |
| W2 D6–7 | JWT auth basics, deploy to Render/Vercel | Deploy full-stack app live |

### 1 Month

| Week | Topics | Project |
|------|--------|---------|
| 1 | HTML/CSS/JS + Git/GitHub | Portfolio website |
| 2 | ES6+, async JS, React fundamentals | Weather app (API) |
| 3 | React Router, forms, state management | Blog frontend |
| 4 | Node/Express, REST, MongoDB, auth, deploy | Blog CMS (MERN mini) |

### 3 Months

| Month | Topics | Projects |
|-------|--------|----------|
| M1 | Frontend mastery: HTML/CSS/JS/React/Tailwind | Portfolio + To-Do + Weather app |
| M2 | Backend: Node/Express, PostgreSQL/MongoDB, REST, JWT | Task manager API + Blog CMS |
| M3 | Full-stack integration, Spring Boot intro, payments, CI/CD basics, capstone | **Capstone: E-commerce or Job Portal (MERN)** |

### 6 Months

| Month | Topics | Projects |
|-------|--------|----------|
| M1 | HTML5, CSS3, JS ES6+, DOM, responsive design, accessibility | Portfolio + Calculator |
| M2 | Advanced JS, async, React (hooks, context, router) | To-Do + Weather + Blog UI |
| M3 | React advanced, Tailwind, testing (Jest), deployment | Full React SPA with routing |
| M4 | Node.js, Express, REST APIs, error handling, file upload | REST API + Auth backend |
| M5 | MongoDB + PostgreSQL, Mongoose, SQL joins, indexing | Database-driven app |
| M6 | Spring Boot, JWT/OAuth, payment gateway, Docker basics, capstone | **Capstone: E-commerce / Social Media Clone / Job Portal** |

### 9 Months — Advanced

Everything in 6 months **plus:** TypeScript, Next.js, microservices basics, Redis caching, WebSockets, GraphQL intro, AWS/Vercel pipelines, 2 capstones (SaaS product + Enterprise RBAC dashboard), open-source sprint, mock interviews + system design basics.
