## Package Details

### Quick Start (2 Weeks)

- **Includes:** One track crash course + intro career module
- **Outcome:** Skill awareness; sample project for portfolio
- **Ideal for:** Professionals exploring a new domain before committing

### Foundation (1 Month)

- **Includes:** One track foundation + resume/LinkedIn basics
- **Outcome:** 2 guided projects; GitHub portfolio started
- **Ideal for:** School/college students and career switchers

### Job Ready (3 Months)

- **Pick one primary track:**
  - [Full Stack Development](tracks/01-full-stack-development.md)
  - [Mobile App Development](tracks/04-mobile-app-development.md)
  - [AI, ML & Data Science](tracks/05-ai-ml-data-science.md)
  - [Cybersecurity & Ethical Hacking](tracks/06-cybersecurity-ethical-hacking.md)
- **Outcome:** Internship-ready with capstone project
- **Ideal for:** Final-year students seeking internships

### Professional (6 Months)

**Option A — Full Stack Path**
| Month | Primary Track | Supporting Track |
|-------|---------------|----------------|
| M1–M4 | Full Stack (M1–M4) | — |
| M5 | Full Stack (M5) | Database Technologies (M2 topics) |
| M6 | Full Stack Capstone | DevOps basics (Docker, CI/CD) |

**Option B — Data & AI Path**
| Month | Primary Track | Supporting Track |
|-------|---------------|----------------|
| M1–M4 | AI/ML & Data Science (M1–M4) | Database (SQL analytics) |
| M5–M6 | AI/ML Capstone | Cloud deployment basics |

- **Outcome:** Job-ready with production capstone
- **Ideal for:** Graduates targeting placement drives

### Expert (9 Months)

**Option A — Full Stack + Cloud + DevOps**
| Phase | Months | Tracks |
|-------|--------|--------|
| Foundation | M1–M3 | Full Stack (3-month tier) |
| Backend depth | M4–M5 | Database + Programming Languages (DSA) |
| Cloud & DevOps | M6–M7 | Cloud Platforms + DevOps |
| Capstones | M8–M9 | 2 advanced capstones + certification prep |

**Option B — AI/ML + Cloud + MLOps**
| Phase | Months | Tracks |
|-------|--------|--------|
| Data foundation | M1–M3 | AI/ML (3-month tier) |
| Deep learning | M4–M5 | AI/ML (M4–M5) |
| MLOps & Cloud | M6–M7 | Cloud Platforms + DevOps |
| Capstones | M8–M9 | Production ML pipeline + AI product |

- **Outcome:** Industry-ready with certifications
- **Ideal for:** Serious career builders targeting mid-level roles

### Complete Tech (9 Months — Staggered)

| Month | Focus Track | Hours/Week |
|-------|-------------|------------|
| M1 | Full Stack (frontend) | 20 |
| M2 | Full Stack (backend) | 20 |
| M3 | Database Technologies | 15 |
| M4 | Programming Languages (DSA) | 15 |
| M5 | Cloud & DevOps | 15 |
| M6 | AI/ML intro | 15 |
| M7 | Cybersecurity basics OR Mobile | 15 |
| M8 | UI/UX OR Digital Marketing (elective) | 10 |
| M9 | Capstone + placement sprint | 25 |

- **Outcome:** Broad IT competency across domains
- **Ideal for:** Students wanting maximum exposure before specialization
