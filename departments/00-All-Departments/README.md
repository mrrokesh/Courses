| Tier | Folder | Priority |
|------|--------|----------|
| 1 | `tier-1/` | Highest demand |
| 2 | `tier-2/` | Strong demand |
| 3 | `tier-3/` | Specialized / niche-high-value |

- **tier-1/** - 3 course(s)
- **tier-3/** - 1 course(s)
