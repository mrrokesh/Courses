## Service Overview

**Who this is for:** Every enrolled student across all technical tracks (Tracks 1–11).

**Integration model:** Career modules run on a fixed schedule based on program duration — not as optional add-ons.

**Goal:** Transform technical skills into job offers through communication, portfolio, and interview readiness.

## Module 1: Self-Introduction & Communication (All Tracks, Week 1)

**Activities:**
- Record 2-minute self-introduction video
- Peer review using feedback rubric
- Professional email writing exercise
- Active listening workshop

**Deliverables:**
- [ ] Intro video (recorded and reviewed)
- [ ] Written bio (100 words) for LinkedIn/GitHub
- [ ] Professional email to mentor (subject line, greeting, body, sign-off)

**Assessment rubric:**

| Criterion | Points |
|-----------|--------|
| Clarity and structure | 25 |
| Confidence and delivery | 25 |
| Relevant skills mentioned | 25 |
| Professional tone | 25 |

## Module 2: Resume + LinkedIn (1 Month+)

**Timing:** Week 2–3 (1-month) | Month 1 (3-month+)

**Activities:**
- ATS-optimized resume workshop (1-page rule for juniors)
- LinkedIn profile audit and optimization
- Action verb and quantified achievement exercise
- Headline and summary writing

**Deliverables:**
- [ ] Resume (PDF) passing ATS scan (Jobscan or mentor review)
- [ ] LinkedIn profile with: professional photo, headline, summary, 3+ skills, project links
- [ ] Resume tailored for one target role (Full Stack / Data / etc.)

**Resume template sections:**
1. Contact info
2. Summary (2–3 lines)
3. Skills (grouped by category)
4. Projects (3+ with tech stack and outcome)
5. Education
6. Certifications (if any)

## Module 3: Portfolio + GitHub (3 Months+)

**Timing:** Month 2 (3-month) | Month 2–3 (6-month+)

**Activities:**
- GitHub profile README optimization
- Project case study writing (problem → solution → result)
- Portfolio website review (optional: deploy on GitHub Pages)
- Code review etiquette and contributing guidelines

**Deliverables:**
- [ ] GitHub profile with pinned repos (3+)
- [ ] Each project README includes: description, stack, screenshots, live demo, learnings
- [ ] One written case study (500+ words) for best project

**See also:** [GitHub Portfolio Requirements](../00-duration-framework.md#github-portfolio-requirements)

## Module 4: Interview Prep (6 Months+)

**Timing:** Month 4–6 (6-month, weekly) | Month 4–8 (9-month, weekly)

**Activities:**
- Aptitude practice (logical, quantitative, verbal) — 2 hrs/week
- Technical mock interviews (track-specific) — 1 hr/week
- Behavioral questions (STAR method) — bi-weekly
- System design intro (9-month only) — 2 sessions/month

**Weekly schedule (6-month+):**

| Day | Activity |
|-----|----------|
| Tuesday | Aptitude set (10 problems) |
| Thursday | Technical mock (45 min + feedback) |
| Saturday | Behavioral STAR practice (2 questions) |

**Deliverables:**
- [ ] Aptitude log (100+ problems over program)
- [ ] 8+ technical mock interviews completed
- [ ] STAR stories documented (5 situations)
- [ ] Mock interview score improvement tracked

**Track-specific interview topics:**

| Track | Focus Areas |
|-------|-------------|
| Full Stack | JS/React/Node, REST, auth, SQL vs NoSQL |
| Database | Normalization, indexing, query optimization |
| Programming | DSA, OOP, complexity analysis |
| Mobile | Flutter/RN lifecycle, state management |
| AI/ML | ML pipeline, metrics, overfitting, deployment |
| Cybersecurity | OWASP, pentest methodology, tools |
| DevOps | Docker, K8s, CI/CD, monitoring |
| Cloud | AWS services, architecture, IAM |
| UI/UX | Design process, portfolio walkthrough |
| Digital Marketing | Campaign metrics, SEO, platform tools |

## Module 5: Placement Sprint (9 Months)

**Timing:** Month 8–9

**Activities:**
- Company-targeted application strategy (50+ applications)
- HR mock interviews (salary negotiation, culture fit)
- Technical panel mock (2 interviewers, 60 min)
- Offer evaluation and decision framework

**Deliverables:**
- [ ] Application tracker spreadsheet (company, role, date, status)
- [ ] 50+ applications submitted
- [ ] 5+ HR mock sessions completed
- [ ] 3+ technical panel mocks completed
- [ ] Offer comparison matrix (if multiple offers)

**Application tracker columns:**
Company | Role | Date Applied | Status | Follow-up | Notes

## Appendix: Mentor Notes

**Common pitfalls:** Generic resume with no projects; empty GitHub; skipping mock interviews; applying without tailoring resume.

**Resources:** LinkedIn Learning (interview prep), Pramp (peer mocks), LeetCode (technical), IndiaBIX (aptitude).

**Cross-track integration:** Every technical track file references this track; mentors must schedule career modules alongside technical labs per the integration matrix above.

**Pairs with:** [Track 12 — School Projects](12-school-college-projects.md) for viva prep; [Track 13 — Thesis Writing](13-project-report-thesis-writing.md) for presentation skills.
