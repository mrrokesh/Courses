**Department:** ECE | **MR ROKESH TECHNOLOGY**

</div>

## Track Overview

**Who this is for:** ECE students targeting VLSI design, FPGA development, and digital IC design roles.

**Prerequisites:** Digital electronics fundamentals; logic gates, flip-flops.

**Tools:** Xilinx Vivado / Intel Quartus, Verilog/SystemVerilog, ModelSim, FPGA boards (Basys/Nexys), Logisim (intro).

---

## 2-Week Crash Course

| Days | Topics | Deliverable |
|------|--------|-------------|
| D1–2 | Combinational logic, K-maps, multiplexers, adders | Logic design exercises |
| D3–4 | Sequential circuits: FFs, counters, FSMs | FSM design (traffic light) |
| D5–6 | Verilog: modules, always blocks, testbenches | Simulated Verilog designs |
| D7–8 | FPGA synthesis, constraints, pin assignment | FPGA blink + counter |
| D9–10 | Capstone | **FPGA project on board** |

---

## 1-Month Foundation

| Week | Topics | Project |
|------|--------|---------|
| W1 | Advanced Verilog, hierarchical design | ALU design |
| W2 | FSM design, UART/I2C intro (RTL) | Serial communication module |
| W3 | FPGA IP cores, block RAM, FIFOs | Buffer system design |
| W4 | Timing constraints, simulation vs synthesis | **Complete FPGA system** |

## 3-Month Job Ready

| Month | Topics | Projects |
|-------|--------|----------|
| M1 | RTL design best practices, linting | UART + SPI controller |
| M2 | Pipelining, memory interfaces | Simple processor datapath |
| M3 | ASIC design flow intro, capstone | **RISC processor or DSP block capstone** |

## 6-Month Professional

Add: SystemVerilog, UVM intro, static timing analysis, low-power design basics, 3 RTL projects, industry coding standards.

## Appendix: Mentor Notes
