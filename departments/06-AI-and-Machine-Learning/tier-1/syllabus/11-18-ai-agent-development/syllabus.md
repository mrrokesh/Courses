## Track Overview

**Who this is for:** Developers building autonomous AI agents, copilots, and workflow automation systems.

**Prerequisites:** Python, REST APIs, basic [Prompt Engineering](17-prompt-engineering.md) (or concurrent enrollment).

**Tools:** OpenAI/Anthropic APIs, LangChain, LangGraph, CrewAI/AutoGen, Python, FastAPI, vector DBs, browser/API tools.

## 2-Week Crash Course

| Days | Topics | Deliverable |
|------|--------|-------------|
| D1–2 | Agent concepts: tools, memory, planning loops | Architecture diagram |
