# Track 16: Data Analytics

**MR ROKESH TECHNOLOGY — Training & Internship Programs**

</div>

> **Skill path:** Excel → SQL → Python/Pandas → Statistics → Visualization → Dashboards → Business Intelligence

**Related:** [Track 5 — AI/ML](05-ai-ml-data-science.md) | [Track 29 — Power BI](29-power-bi.md) | [Track 30 — Tableau](30-tableau.md)

## Track Overview

**Who this is for:** Aspiring data analysts, business analysts, and professionals needing data-driven decision skills.

**Prerequisites:** Basic computer literacy; no coding required for 2-week/1-month tiers.

**Tools:** Excel, Google Sheets, SQL (PostgreSQL/MySQL), Python, Pandas, Jupyter, Matplotlib/Seaborn, Power BI or Tableau (intro).

**Certification targets:** Google Data Analytics Certificate alignment (optional, 6-month)

## 2-Week Crash Course

**Mini Project:** Sales analytics dashboard | **Level:** L1

| Days | Topics | Deliverable |
|------|--------|-------------|
| D1–2 | Excel: formulas, pivot tables, charts | Sales summary dashboard |
| D3–4 | SQL: SELECT, JOIN, GROUP BY, aggregations | 10 analytical queries |
| D5–6 | Python + Pandas: data cleaning, EDA | Cleaned dataset + insights report |
| D7–8 | Visualization: Matplotlib/Seaborn or BI tool | Interactive charts |
| D9–10 | Storytelling with data, capstone | **Executive analytics report** |

## 1-Month Foundation

| Week | Topics | Project |
|------|--------|---------|

## 3-Month Job Ready

| Month | Topics | Projects |
|-------|--------|----------|
