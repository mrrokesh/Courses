**MR ROKESH TECHNOLOGY — Training & Internship Programs**

</div>

**Related:** [Track 19 — Robotics & IoT](19-robotics-iot.md) | [Track 20 — Embedded Systems](20-embedded-systems.md)

## Track Overview

**Who this is for:** Electrical, instrumentation, and automation engineers targeting manufacturing and process industries.

**Prerequisites:** Basic electrical knowledge; digital logic helpful.

**Tools:** Siemens TIA Portal / Allen-Bradley Studio 5000 (or openPLC), Factory I/O simulator, HMI panels, SCADA (WinCC/industrialSoft intro).

## 2-Week Crash Course

| Days | Topics | Deliverable |
|------|--------|-------------|
| D1–2 | Industrial automation overview, safety, I/O types | Safety documentation |
| D3–4 | Ladder logic: contacts, coils, timers, counters | 5 ladder programs |
| D5–6 | PLC programming (simulator) | Motor start/stop circuit |
| D7–8 | Sensors: proximity, photoelectric, encoders | Sensor integration lab |
| D9–10 | Factory I/O simulation | **Conveyor sorting system** |

## Appendix: Mentor Notes
