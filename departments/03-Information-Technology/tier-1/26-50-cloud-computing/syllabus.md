## Track Overview

**Who this is for:** CSE and IT students targeting cloud engineer, cloud administrator, and solutions architect roles. Consolidated "Cloud Computing" program for Tier 1 marketing.

**Prerequisites:** Basic Linux and networking helpful; taught from fundamentals.

**Tools:** AWS, Azure, GCP free tiers, CLI tools, Docker, Terraform intro, CloudWatch/Monitor.

**Certification targets:** AWS Cloud Practitioner / SAA (6-month), AZ-900 (optional)

## 2-Week Crash Course

| Days | Topics | Deliverable |
|------|--------|-------------|
| D1–2 | Cloud models, IAM, EC2/VMs | Running cloud instance |
| D3–4 | S3/Blob storage, CDN, static sites | Hosted static website |
| D5–6 | RDS/SQL, VPC/networking basics | 2-tier database app |
| D7–8 | Lambda/Functions, API Gateway | Serverless API |
| D9–10 | Cost optimization, capstone | **3-tier cloud architecture live** |

## 1-Month Foundation

| Week | Topics | Project |
|------|--------|---------|
| W1 | AWS core: IAM, EC2, S3, VPC | Secure VPC setup |
| W2 | Azure OR GCP parallel lab | Second platform deployment |
| W3 | Containers on cloud (ECS/Cloud Run) | Containerized app deploy |
| W4 | Monitoring, billing alerts, capstone | **Full cloud project** |

## 3-Month Job Ready

| Month | Topics | Projects |
