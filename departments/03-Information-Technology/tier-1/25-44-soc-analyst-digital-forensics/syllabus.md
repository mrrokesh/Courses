**Departments:** Cyber Security · IT | **MR ROKESH TECHNOLOGY**

</div>

## Track Overview

**Who this is for:** Cyber Security department students targeting blue team, SOC analyst, and digital forensics roles.

**Prerequisites:** [Track 6](06-cybersecurity-ethical-hacking.md) 1-month foundation OR [Track 41](41-network-system-administration.md) networking basics.

**Tools:** Splunk/Wazuh/Elastic SIEM, Wireshark, Autopsy, FTK intro, VirusTotal, MITRE ATT&CK, isolated forensic VMs.

## 2-Week Crash Course

| Days | Topics | Deliverable |
|------|--------|-------------|
| D1–2 | SOC roles, kill chain, MITRE ATT&CK | ATT&CK mapping exercise |
| D3–4 | SIEM: log ingestion, dashboards, alerts | 5 alert triage reports |
| D5–6 | Network forensics, PCAP analysis | Wireshark investigation report |
| D7–8 | Disk forensics intro, Autopsy, evidence handling | Forensic imaging lab |
| D9–10 | Incident response lifecycle, capstone | **IR report on simulated breach** |
