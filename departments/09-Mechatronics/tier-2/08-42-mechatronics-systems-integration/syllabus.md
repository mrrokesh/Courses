## Track Overview

**Who this is for:** Mechatronics students integrating mechanical, electrical, and software subsystems into complete automated systems.

**Prerequisites:** Basic physics and programming; [Track 22 Arduino](22-arduino.md) or equivalent helpful.

**Tools:** SolidWorks/Fusion 360, Arduino/STM32, PLC simulator, ROS2, MATLAB/Simulink, pneumatic/electro-pneumatic kits.

## 2-Week Crash Course

| Days | Topics | Deliverable |
|------|--------|-------------|
| D1–2 | Mechatronics system overview, sensors, actuators | System block diagram |
| D3–4 | Arduino/embedded control of motors, relays | Motor control circuit |
| D5–6 | Pneumatics/electro-pneumatics basics | Pneumatic circuit + control |
| D7–8 | PLC ladder integration with sensors | PLC-controlled station |
| D9–10 | Capstone integration | **Automated sorting/conveyor station** |
