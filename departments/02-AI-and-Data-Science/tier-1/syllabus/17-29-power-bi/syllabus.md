**MR ROKESH TECHNOLOGY — Training & Internship Programs**

## Track Overview

**Who this is for:** Data analysts, business professionals, and BI developers in Microsoft-centric organizations.

**Prerequisites:** Basic Excel; SQL helpful for 1-month+ tiers.

**Tools:** Power BI Desktop, Power Query, DAX, Power BI Service, Excel integration.

**Certification targets:** PL-300 Microsoft Power BI Data Analyst (6-month)

## 2-Week Crash Course

| Days | Topics | Deliverable |
|------|--------|-------------|
| D1–2 | Power BI interface, data import, relationships | Connected data model |
| D3–4 | Power Query: transform, merge, append | Cleaned dataset |
| D5–6 | Visualizations: charts, maps, KPIs, slicers | 5-page report draft |
| D7–8 | DAX basics: CALCULATE, SUM, COUNT, filters | 10 DAX measures |
| D9–10 | Publishing, sharing, capstone | **Published sales dashboard** |
