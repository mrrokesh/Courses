## Track Overview

**Who this is for:** Aspiring DevOps engineers, SREs, platform engineers, and backend developers moving into infrastructure automation.

**Prerequisites:** Basic Linux and scripting helpful; programming basics (Python/Bash) recommended for 3-month+ tiers.

**Tools:** Ubuntu/Linux, Git, GitHub Actions, Jenkins, Docker, Kubernetes, Terraform, Ansible, Prometheus, Grafana, ArgoCD, Helm.

**Certification targets:** CKA, CKAD, AWS DevOps Engineer (optional, 9-month)

## 2-Week Crash Course

**Mini Project:** Containerized app with CI/CD pipeline | **Level:** L1

| Days | Topics | Lab | Deliverable |
|------|--------|-----|-------------|
| D1–2 | Linux admin, SSH, Bash scripting | Automation scripts | 3 Bash utility scripts |
| D3–4 | Git workflows, branching, GitHub Actions | CI pipeline | Build + test on push |
| D5–6 | Docker, Dockerfile, Compose | Containerize web app | Multi-container setup |
| D7–8 | K8s basics: pods, services, deployments | Minikube deploy | App running on K8s |
| D9–10 | Monitoring intro, deploy live | **CI/CD + deployed container app** | Live URL + green pipeline |

---

## 1-Month Foundation

| Week | Topics | Project |
|------|--------|---------|
| W1 | Linux, shell scripting, Git | Backup + log rotation scripts |
| W2 | Docker, Compose, registry | 3-tier containerized app |
| W3 | Kubernetes, Helm charts | Deploy with Helm on Minikube |
| W4 | CI/CD (GitHub Actions/Jenkins), secrets | **Full pipeline to K8s** |

**Capstone acceptance:** CI runs tests; Docker image pushed; K8s deployment automated; secrets not in Git.

## 3-Month Job Ready

| Month | Topics | Projects |
