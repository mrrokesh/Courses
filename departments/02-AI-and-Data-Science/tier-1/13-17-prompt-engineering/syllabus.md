## Track Overview

**Who this is for:** Developers, AI engineers, and technical professionals building LLM-powered applications.

**Prerequisites:** Basic Python; familiarity with APIs helpful. For non-technical audience see [Track 33](33-prompt-engineering-for-business.md).

**Tools:** OpenAI API, Anthropic API, LangChain, LlamaIndex, Python, Jupyter, vector DBs (Chroma/Pinecone), prompt evaluation frameworks.

**Certification targets:** Portfolio of production prompt systems (9-month)

## 2-Week Crash Course

**Mini Project:** Prompt library + API chatbot | **Level:** L1

| Days | Topics | Deliverable |
|------|--------|-------------|
| D1–2 | LLM fundamentals, tokens, temperature, system prompts | Prompt cheat sheet |
| D3–4 | Zero-shot, few-shot, role prompting | 20 tested prompt templates |
| D5–6 | Chain-of-thought, structured output (JSON) | Structured extraction demo |
| D7–8 | OpenAI/Anthropic API integration | Python chatbot script |
| D9–10 | Evaluation basics, iteration | **Documented prompt library + demo** |
