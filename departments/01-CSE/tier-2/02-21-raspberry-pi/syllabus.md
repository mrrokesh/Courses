## Track Overview

**Who this is for:** Makers, IoT developers, and students building edge computing and smart device projects.

**Prerequisites:** Basic Python; Linux command line taught from Week 1.

**Tools:** Raspberry Pi 4/5, Raspberry Pi OS, GPIO library (gpiozero/RPI.GPIO), Pi Camera, MQTT, Flask/FastAPI.

**Hardware:** Raspberry Pi board + power supply + microSD + sensor kit
