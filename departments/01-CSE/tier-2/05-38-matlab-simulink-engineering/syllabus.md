## Track Overview

**Who this is for:** Engineering students across EEE, ECE, Mechanical, and AI branches needing simulation and numerical computing skills.

**Prerequisites:** Basic calculus and physics; programming helpful but taught from basics.

**Tools:** MATLAB, Simulink, Control System Toolbox, Signal Processing Toolbox, student license.

## 2-Week Crash Course

| Days | Topics | Deliverable |
|------|--------|-------------|
| D1–2 | Variables, matrices, functions, plotting | 5 MATLAB scripts |
| D3–4 | Symbolic math, ODE solvers, curve fitting | Data analysis report |
| D5–6 | Simulink basics: blocks, transfer functions | First Simulink model |
| D7–8 | PID control, step response, tuning | Motor control simulation |
| D9–10 | Capstone | **Simulink project + report** |

---

## 1-Month Foundation

| Week | Topics | Project |
|------|--------|---------|
| W1 | MATLAB programming, file I/O, debugging | Numerical methods toolkit |
| W2 | Simulink: subsystems, S-functions intro | Multi-block simulation |
| W3 | Control systems: Bode, root locus, stability | Feedback control design |
| W4 | Signal processing: FFT, filtering | **Signal analysis project** |
