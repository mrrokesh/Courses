## Track Overview

**Who this is for:** Students preparing for coding interviews, competitive programming, backend roles, or language specialization.

**Prerequisites:** Basic computer literacy. 2-week crash requires picking **one primary language**.

**Language paths:**
- **Python** — Data science crossover, rapid prototyping, interviews
- **Java** — Enterprise, Android crossover, OOP depth
- **C/C++** — Systems programming (6-month+ systems track)
- **Dart** — Flutter crossover (see [Track 4](04-mobile-app-development.md))
- **JavaScript** — Full-stack crossover (see [Track 1](01-full-stack-development.md))

**Tools:** VS Code, language SDK (Python 3.11+, JDK 17+, GCC), LeetCode/HackerRank accounts, Git/GitHub.

**Certification targets (9-month):** Oracle Java Foundations (optional); interview readiness primary goal

## 2-Week Crash Course (Pick 1 Primary Language)

**Mini Project:** Calculator / Quiz app | **Level:** L1

| Days | Topics | Practice | Self-Check |
|------|--------|----------|------------|
| D1–2 | Syntax, variables, loops, functions, I/O | 20 basic programs | Write FizzBuzz from memory |
| D3–4 | OOP: classes, inheritance, encapsulation, polymorphism | OOP mini library system | Explain inheritance vs composition |
| D5–6 | Arrays, strings, file handling | File-based student records | Big-O of linear search |
| D7–8 | DSA basics: arrays, sorting, searching | 15 LeetCode Easy problems | Implement binary search |
| D9–10 | Recursion, problem-solving patterns | **Calculator / Quiz app** | Trace recursive Fibonacci |

**Language-specific notes:**
- **Python:** Use `dataclasses`, list comprehensions, `with` for file I/O
- **Java:** Emphasize types, access modifiers, `ArrayList`, interfaces
- **C++:** Pointers intro, STL vectors, basic memory concepts
