## Track Overview

**Who this is for:** IT students targeting network administrator, system administrator, and IT infrastructure roles.

**Prerequisites:** Basic computer literacy; Linux introduced from Week 1.

**Tools:** Linux (Ubuntu/CentOS), Windows Server, VirtualBox/VMware, Active Directory, DNS/DHCP, Wireshark, AWS/Azure admin consoles.

**Certification targets:** CompTIA Network+, Linux+ (optional, 6-month)

## 2-Week Crash Course

| Days | Topics | Deliverable |
|------|--------|-------------|
| D1–2 | OSI/TCP-IP, IP addressing, subnetting | Subnetting worksheet |
| D3–4 | Linux admin: users, permissions, services | Configured Linux server |
| D5–6 | Windows Server, Active Directory basics | AD domain with 3 users |
| D7–8 | DNS, DHCP, file sharing, SSH | Working lab network |
| D9–10 | Wireshark, troubleshooting, capstone | **Network documentation + lab demo** |
