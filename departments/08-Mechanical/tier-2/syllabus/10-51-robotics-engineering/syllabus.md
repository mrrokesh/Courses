## Track Overview

**Who this is for:** Mechatronics, Mechanical, and ECE students targeting robotics engineer roles (distinct from general IoT track).

**Prerequisites:** Python/C++; basic mechanics; [Track 22 Arduino](22-arduino.md) helpful.

**Tools:** ROS2, Gazebo, RViz2, Python/C++, motor drivers, LIDAR/camera sensors, TurtleBot or custom kit.

---

## 2-Week Crash Course

| Days | Topics | Deliverable |
|------|--------|-------------|
| D1–2 | Robot anatomy, DOF, kinematics intro | Robot specification doc |
| D3–4 | Sensors: encoder, IMU, LIDAR, camera | Sensor integration lab |
| D5–6 | Actuators: DC/servo/stepper, motor drivers | Motor control system |
| D7–8 | ROS2 nodes, topics, Gazebo simulation | Simulated robot movement |
| D9–10 | Capstone | **Autonomous line-follow or obstacle avoid bot** |

---

## 1-Month Foundation

| Week | Topics | Project |
|------|--------|---------|
| W1 | Kinematics, forward/inverse intro, URDF | Custom robot model |
| W2 | ROS2 middleware, publishers/subscribers | Sensor data pipeline |
| W3 | SLAM intro, mapping, localization | Generated environment map |
| W4 | Nav2 stack, path planning | **Point-to-point autonomous robot** |

## 3-Month Job Ready
