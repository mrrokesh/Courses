# MR ROKESH TECHNOLOGY

**College Event Day Checklist**

</div>

---

## Before Event (1 Week Prior)

- [ ] Confirm date, time, venue with TPO / faculty coordinator
- [ ] Confirm projector, HDMI, microphone, seating capacity
- [ ] Confirm lab access (if hands-on workshop)
- [ ] Print **department poster** (A4 or A3) — qty: 5–10
- [ ] Print **50+ brochures** for interested students
- [ ] Create Google Form + **QR code** (large print for standee)
- [ ] Prepare participation **certificates** (blank with college name field)
- [ ] Test **live demo** offline (backup video if internet fails)
- [ ] Charge laptop, pack HDMI adapter, extension board, mouse
- [ ] Prepare fee structure sheet (2w / 1m / 3m / 6m)
- [ ] Wear branded ID / formal attire

---

## Event Day — Bring

- [ ] Laptop + charger
- [ ] HDMI adapter + extension board
- [ ] Posters + brochures + business cards
- [ ] QR standee or printed QR (A4 minimum)
- [ ] Certificates + pens
- [ ] Demo hardware (if ECE/Mech/Fashion): Arduino, fabric swatch, etc.
- [ ] MoU draft (if HOD meeting scheduled)
- [ ] Notepad for HOD/TPO feedback

---

## During Event

| Time | Action |
|------|--------|
| **-30 min** | Arrive, test projector, set up QR standee |
| **Start** | Introduce MR ROKESH + department topic |
| **Mid** | Live demo (most important segment) |
| **Break** | Push QR registration — stand at exit |
| **End** | CTA: free career assessment + early-bird deadline |
| **After** | Photo with HOD/TPO (for social media) |
| **After** | Collect coordinator WhatsApp for follow-up |

---

## Lead Collection (Minimum Fields)

- Full Name
- Department & Year
- Phone / WhatsApp
- Email
- Course interest (dropdown)
- Attended seminar? Y/N

---

## Within 24 Hours Post-Event

- [ ] WhatsApp thank-you to coordinator
- [ ] Email thank-you + enrollment details (Template 4)
- [ ] Message all registered leads on WhatsApp
- [ ] Post event photo on Instagram/LinkedIn (with college permission)
- [ ] Log college in outreach tracker: date, dept, attendees, leads, follow-up date

---

## Within 7 Days

- [ ] Call top 20 interested leads
- [ ] Offer early-bird batch discount (deadline date)
- [ ] Schedule follow-up visit or bootcamp if HOD interested
- [ ] Propose MoU if 30+ students registered interest

---

## Conversion Targets

| Event Size | Target Leads | Target Enrollments |
|------------|-------------|-------------------|
| 50 students | 15+ registrations | 3–5 paid |
| 100 students | 30+ registrations | 8–12 paid |
| 200+ students | 50+ registrations | 15–25 paid |

---

## Emergency Backup

- [ ] Demo video on USB if live demo fails
- [ ] Mobile hotspot if college Wi-Fi fails
- [ ] PDF slides on USB (not cloud-only)
